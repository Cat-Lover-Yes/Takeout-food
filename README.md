## 🔷使用技术

## 项目部署

1. 下载本项目到服务器。

2. 修改`src/main/resources/application.yml`文件内的数据库信息
> 需要改动的就是下面我勾选的三个地方

![](resource/截屏2022-11-28 21.59.36.png)

  - 数据库名
  - 数据库用户名
  - 数据库密码
3. 在服务器上部署时，将项目打成`jar`包，使用`java -jar 包名`进行运行
4. 访问localhost:8080/backend/index.html即可
